# Plantainance - Prototype MVP

Plantainance is a subscription-based plant maintenance service designed for busy urban lifestyles. We care for your plants, so you don't have to.

This repository contains the code for our MVP prototype website to validate market demand.

## Project Structure
```
plantainance/
├── index.html        # Main landing page
├── style.css         # Custom premium styling
├── assets/           # Directory for future local images, logos, and favicon
└── README.md         # Project documentation
```

## Features
- **Hero Section**: Clear value proposition with a "Choose a Plan" call to action.
- **Services Showcase**: Details the core offerings (Watering, Pruning, Pest Control, etc.).
- **Pricing Plans**: Three-tier flexible subscription plans (Basic, Standard, Premium).
- **Lead Capture Form**: Embedded contact form to book first visits and measure customer interest.

## Deployment Instructions (GitHub Pages)
The site is designed to be hosted for free using GitHub Pages:
1. Commit all files (`index.html`, `style.css`, etc.) to the `main` branch of your GitHub repository.
2. In your repository, go to **Settings** -> **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Set the branch to `main` and folder to `/(root)`.
5. Click **Save**. Your site will be published at `https://<your-username>.github.io/plantainance`.

## Tech Stack
- **Frontend**: Standard HTML5 and Vanilla CSS3
- **Icons**: Phosphor Icons (via CDN)
- **Imagery**: High-quality placeholders sourced from Unsplash
